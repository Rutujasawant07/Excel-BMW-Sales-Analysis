## Excel BMW Sales Analysis
Sales data analysis of Mahindra vehicles using Excel to identify revenue trends, top-performing models, regional performance, and monthly growth insights.

## 🚗 BMW Sales Analysis (Excel Project)
 
## 📌 Project Overview

This project focuses on analyzing BMW vehicle sales data using Microsoft Excel to uncover key business insights, track performance trends, and support data-driven decision-making. The dashboard provides a clear view of revenue patterns, model performance, and regional sales distribution.

## 🛠 Tools Used

• Microsoft Excel
• Pivot Tables & Pivot Charts
• Data Cleaning & Transformation
• Conditional Formatting
• Excel Formulas (IF, SUMIFS, VLOOKUP/XLOOKUP)
• Interactive Dashboard Elements (Slicers, Timelines)

## 📊 Key Analysis Performed
• Monthly Sales Trend Analysis
• Region-wise Sales Performance
• Top-Selling BMW Models
• Revenue Performance Insights
• Growth Percentage Calculation

## 📈 Insights Derived
 • SUV models (X series) are the top revenue contributors
 • Sales show peak growth in festive and year-end months
 • Europe and North America are the strongest performing regions
 • Certain sedan models show stable but slower growth compared to SUVs
 • Monthly sales indicate fluctuations due to seasonality and promotions
 • Premium models generate higher revenue despite lower unit sales

## 📊 Dashboard Features
• Interactive slicers for Region, Model, and Time Period
• Dynamic charts 
• KPI indicators 
• Clean and user-friendly layout for quick decision-making

## 🎯 Conclusion
The project demonstrates how Excel can be used as a powerful analytical tool to transform raw sales data into meaningful business insights.
